Design notes

Same content as the current Schedule Today view, restyled to match BillingDashboardV2. Uses identical tokens (cream, forest, sage, gold, rose), identical typography (Cormorant Garamond serif for headings + giant numerals, system sans for body), identical period-tab pattern (equal-width pills on cream-deep background), and identical card-radius (18px) plus shadow-soft.

What changed

  • Period tabs swapped from cool-gray #F3F4F6 background to --cream-deep. Active text from #1F2937 to --forest-deep. Tabs now equal-width (flex: 1) like Billing.
  • Yearly tab added, bringing parity with Billing's 5-tab pattern.
  • Action buttons moved below the tab bar per your direction. Same three actions (Book, Previews, Time off), now styled as soft white cards with cream borders.
  • Stat strip from the old design (0 Today, 0 Brief ready, 0 Need intake, 0 This week) is absorbed into the band 2 summary line plus band 3 attention list.
  • Hero band introduces "Next up" with giant serif time, client, service, countdown pill, and two inline actions.
  • Shape band renders the day as a horizontal timeline strip with appointment blocks, an animated "now" marker, and hour labels below.
  • Attention band only renders when there's something to resolve.
  • All sessions list uses session-row format matching Billing's ReceiptCard density.

What stayed the same

  • All four bands of content (next session, daily summary, attention items, full session list) are preserved.
  • Tap behavior, deep links, modal opens are unchanged.
  • Time-rail TimelineView component can either stay as-is below this dashboard, or this dashboard replaces it. Open question.

Open questions for HK

  • Does this replace the existing TimelineView or sit above it? If above, do we hide the vertical time-rail under a "View timeline" expand?
  • Counters on the action buttons. The old strip showed "0 Today | 0 Brief ready | 0 Need intake | 0 This week", and those numbers are now buried in band 2's summary line. Should we surface them as a small ribbon below the action row?
  • Trust line at bottom. "Your day, in one calm picture." Want a different line, or none?
